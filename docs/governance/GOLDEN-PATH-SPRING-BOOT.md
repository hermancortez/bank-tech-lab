# Golden Path Spring Boot

1. Java 21, Maven Wrapper y Spring Boot.
2. Límites hexagonales: dominio, aplicación y adaptadores.
3. Validation, Problem Details, Actuator y pruebas.
4. Imagen no-root, health probes y configuración externa.
5. Pipeline con build, test, análisis, escaneo y despliegue.

Es una ruta recomendada, no una jaula: toda excepción se registra con motivo y responsable.
