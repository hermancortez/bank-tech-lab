# account-service

## Responsabilidad

Gestiona cuentas bancarias asociadas a clientes.

## Casos de uso

- Crear cuenta.
- Listar cuentas.
- Buscar cuenta por id.
- Listar cuentas por cliente.

## API local

```text
POST /accounts
GET /accounts
GET /accounts/{id}
GET /accounts/customer/{customerId}
```
