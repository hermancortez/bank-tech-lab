# Git Flow

## Ramas principales

| Rama | Uso |
|------|-----|
| main | Versión estable |
| develop | Integración de cambios |
| feature/* | Nuevas funcionalidades |
| fix/* | Correcciones |
| docs/* | Documentación |
| chore/* | Cambios técnicos menores |

---

## Convención de commits

Ejemplos:

```text
docs: add ports allocation
feat: create customer service
fix: resolve datasource configuration
chore: update docker compose
test: add customer use case tests
refactor: improve domain model
